# dotfiles

個人用 dotfiles リポジトリ。zsh 設定、git 設定、開発ツール設定を含む

## インストール・同期

```bash
sh sync.sh
```

このスクリプトはシンボリックリンクを作成して設定ファイルを適切な場所に配置する

## 含まれる設定

- **zsh** - シェル設定（履歴管理、エイリアス、fzf 統合）
- **git** - グローバル設定とエイリアス
- **starship** - プロンプト設定
- **mise** - ランタイム管理ツール設定
