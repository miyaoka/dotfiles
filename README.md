# dotfiles

個人用dotfilesリポジトリ。zsh設定、git設定、開発ツール設定を含む

## インストール

```bash
sh install.sh
```

このスクリプトはシンボリックリンクを作成して設定ファイルを適切な場所に配置する

## 含まれる設定

- **zsh** - シェル設定（履歴管理、エイリアス、fzf統合）
- **git** - グローバル設定とエイリアス
- **starship** - プロンプト設定
- **mise** - ランタイム管理ツール設定
